# Salão Famiglia D'Mano

Site institucional do **Salão Famiglia D'Mano** — barbearia no Lago Sul, Brasília-DF, desde 1981.

## Páginas
- `index.html` — página principal (marca, serviços, barbeiros, FAQ, contato)
- `marcel.html` — perfil do barbeiro Marcel
- `arthur.html` — perfil do barbeiro Arthur
- `naya.html` — perfil da barbeira Naya

Cada perfil traz QR Code Pix para pagamento e link de WhatsApp.

## Publicação
Site estático, sem dependências. Basta hospedar os arquivos (GitHub Pages, Netlify, Vercel, etc.).
